## 시스템 개요

이 시스템은 AI와의 대화 내용과 작업 내역을 자동으로 추적하고 정리합니다.

### 주요 기능
- 작업 수행 시마다 자동으로 `current-session.md`에 기록
- 롤백 발생 시 해당 작업 항목 자동 제거
- 기능 단위 작업 완료 시 제목을 정해 별도 파일로 저장

### 파일 구조
- `{제목}.md`: 기능 단위별 작업 내역 기록
- `current-session.md`: 현재 진행 중인 세션의 작업 내역 (자동 기록)
- `README.md`: 시스템 사용 방법 및 구조 설명
