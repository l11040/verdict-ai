# Walkthrough 추적 시스템

이 폴더는 AI와의 대화 내용과 작업 내역을 정리합니다.

## 구조

- `{제목}.md`: 기능 단위별 작업 내역 기록 (사용자가 정한 제목으로 저장)
- `current-session.md`: 현재 진행 중인 세션의 작업 내역 (임시 파일, 제목을 정할 때까지 사용)

## 사용 방법

1. AI와 대화를 진행하며 작업을 수행합니다.
   - **자동 기록**: 작업을 수행할 때마다 `current-session.md`에 자동으로 기록됩니다.
2. 기능 단위로 작업이 끝나고 작업 내역을 정리하고 싶을 때, 제목과 함께 "작성해줘" 또는 "정리해줘"라고 요청합니다.
   - 예: "로그인 기능 구현에 대한 작업 내역을 작성해줘"
   - 예: "API 엔드포인트 추가 작업을 정리해줘"
3. AI가 대화 내용을 바탕으로 작업 내역을 정리하여 `{제목}.md` 파일로 저장합니다.
4. 롤백된 작업이 있다면 해당 부분은 자동으로 제외됩니다.

> **참고**: 자동 기록 기능은 `.cursorrules` 파일에 정의되어 있으며, 모든 작업이 자동으로 추적됩니다.

## 기록 형식

각 작업은 다음과 같은 형식으로 기록됩니다:

```markdown
## 작업 {번호} - {날짜/시간}

### 요청 내용
{사용자의 요청}

### 수행한 작업
- {작업 1}
- {작업 2}

### 생성/수정된 파일
- {파일 경로}

### 롤백 여부
- [ ] 정상 완료
- [x] 롤백됨 (롤백 시 자동으로 체크)
```
